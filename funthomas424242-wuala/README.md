# Karaf module for Puppet #

This module installs Karaf.

## Description

This module can install Karaf in a simple way.

Currently supports:

* Ubuntu 14.04.

## Usage

General install with puppet in a manifest.pp file:

     class { 'wuala':
           user => 'your_local_login',
           wualaVersion => 'olympus',
           tmpDir => '/tmp',
     }
